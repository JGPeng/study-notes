# 跨端

## 跨端框架
1. React Native：跨web、app端。
    + Facebook 出品；
    + JavaScript语言；
    + JSCore引擎；
    + React设计模式；
    + 原生渲染。
2. WEEX：跨web、android、ios端。
    + Alibaba 出品；
    + JavaScript语言；
    + JS V8引擎；
    + Vue设计模式；
    + 原生渲染。
3. Flutter：跨平台移动UI框架。
    + Google 出品；
    + Dart语言；
    + Flutter Engine引擎；
    + 响应式设计模式；
    + 原生渲染。
